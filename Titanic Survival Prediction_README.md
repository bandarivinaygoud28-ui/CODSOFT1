# Titanic Survival Prediction

## Objective

The objective of this project is to predict whether a passenger survived the Titanic disaster using Machine Learning techniques.

## Dataset

Titanic Dataset

## Technologies Used

* Python
* Pandas
* Scikit-learn

## Algorithm Used

* Random Forest Classifier

## Project Workflow

1. Data Loading
2. Data Cleaning
3. Feature Encoding
4. Train-Test Split
5. Model Training
6. Prediction
7. Evaluation

## Result

The model achieved an accuracy of approximately 82%.

## Conclusion

The Random Forest Classifier successfully predicted passenger survival based on available features such as age, gender, class, and fare.
